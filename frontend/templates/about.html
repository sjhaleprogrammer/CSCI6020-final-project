About the Project

We utilized available datasets to create a model that accurately predicts the outcome of the 2024 U.S Presidential election. Additionally, we predicted the outcome of “swing states” and each individual state. This was done using public datasets from the U.S Government and the latest U.S Census data. By combining this information, we successfully created a model that predicts who will win the U.S presidential election. Once the model was trained, we developed a simple yet effective user interface to display the results.

Training Model Information

Model 1. Logistic Regression (LogisticRegression from sklearn linear_model):
Provides a parametric, interpretable machine learning algorithm used for binary classification tasks. It is particularly effective when the relationship between features and the target is linear or near-linear. Logistic regression assumes independent features and works best when the data is linearly separable.

Model 2. k-Nearest Neighbors (KNeighborsClassifier from sklearn neighbors):
Provides an interpretable, non-parametric approach to classify election outcomes based on historical data. Suitable for cases where relationships between features are complex and non-linear. Target (y_train, y_val) is the dependent variable (Election Result).

Model 3. Decision Tree (DecisionTreeClassifier from sklearn tree):
The Decision Tree model is a non-parametric, tree-based algorithm that splits data recursively based on feature values to classify outcomes. At each split (or "node"), it evaluates a feature and splits the data to maximize information gain, effectively organizing data into branches that lead to classification results.

Model 4. Random Forest (RandomForestClassifier from sklearn ensemble):
The Random Forest model is an ensemble learning algorithm that constructs multiple decision trees during training and merges their predictions for improved accuracy and reduced overfitting. It is particularly effective for classification tasks with non-linear relationships and high-dimensional data.

Model 5. Binary Classification Neural Network (Pytorch is used to define, train, and evaluate the model):
The Binary Classification Neural Network implementation evaluates election results using a neural network trained on processed data. Learns complex, non-linear relationships between numerical and categorical features using multiple layers of computation.
